记录一下 Eric 生前的朋友吧
*不分排名想到谁写谁

绚香 猫
neko 应该是 Eric 在推特中文圈里认识的第一位 MtF
那个时候的 neko 正在美国留学
neko 的 bio 上挂着一大堆奇奇怪怪的标签
但是很善良也很亲近人
是 Eric 在推特上的第一个朋友
当时还认识了飞鸟等人
曾经有一个由三个人组成的小团体
每天打打闹闹
Eric 曾经也对 neko 宣布主权过
后来因为经济原因决定退学回国
家庭压力很大
也被曾经的主人抛弃了
浑浑噩噩过了一段日子后结识了羽毛
就和羽毛在一起每天黏着秀恩爱
但是因为精神原因最后选择离开

羽毛
一位女装生活了很久的小姐姐
有着令人羡慕的长头发
自己一个人在上海上学
会好多语言
也是一个程序员大牛
认识羽毛的契机应该也是在推特
当时被搭讪了
对话内容尽是一些奇奇怪怪的内容
当时以为羽毛是一个大学生
结果后来才知道比 Eric 还小一点
和 neko 在一起了之后一直相互扶持
和两人第一次面基是在徐汇宜家
被羽毛喂了小肉圆
还被拖去哺乳室做了奇奇怪怪的事情
后来得到了一根冰激凌

飞鸟
一开始名字叫阴影
后来感觉不好听于是改名为飞鸟
是一开始的三人组的成员之一
很照顾 neko
在群里面经常会帮着 neko
于是群里一直是二打一的局面
追求过 neko
但是最后不了了之
飞鸟其实平时是个很奇怪的人
但很善良
特别是在自己喜欢的人面前表现更明显
有时候会因为 neko 的事情把 Eric 一顿骂

页页
是在国内 MtF 圈子里认识的
后来才知道是酷安的大佬
是天赋党真的超级好看
对 Eric 很好
但是交流并不算多
和 Eric 一样是 pusheen 的忠实粉丝
家里一大堆 pusheen 的周边

年糕
算是推特中文圈里很有名的人了
因为经常会发色色的东西
内心很脆弱经常容易爆炸
在日本留学
日常活动经常会涉及政治问题
有着一头长发
也是天赋党
一年四季带着猫爪手套
身材特别好
曾经说要和 Eric 一起吃锅包肉
好像后来也忘了

纱路
应该是主动认识 Eric 的
和 Eric 关系很好
当时也是泉的朋友
几乎每天都来陪 Eric 聊天
动不动就要挠痒痒
后来和泉闹矛盾了之后便注销账号了
Eric 也再也没见过她了
其实很想再说说话的可惜找不到了

泉
是后来进入推特中文圈的人
有着自己的小圈子
口头禅就是喵喵叫
内心敏感而且很渴望爱与温暖
因为家庭原因和经济原因选择退学出来打拼
也经历了一段很辛苦的时光
不过现在生活还不错
在上海和朋友居住
日常生活挺快乐的
曾经是 Eric 的 CP
后来 Eric 被 96子 宣布主权后便放弃
现在和咖菲在一起

觉觉
是从羽毛和 neko 那里认识的
是一个计算机大佬
口头禅也是喵喵叫
很害羞很怕生
也是从家里逃出来自己一个人生活的
现在在广州和千岁生活在一起
觉觉和 Eric 一样十分喜欢 pusheen
平时的聊天内容就是互发贴纸
曾经有一个药圈信物
后来转赠给 Eric 了
现在换成一个小 pusheen 挂在包上

nana
应该是在 Eric 最困难的时期照顾 Eric 最多的人了
一开始在推特上认识
每天都会跑来找 Eric 聊天
nana 在澳大利亚上班
后来回国后和 Eric 面基
是一个扎着小辫子的奇怪大姐姐
喜欢戳 Eric 脸
后来也在 neko 经济有困难的时候帮助过 neko
来过 Eric 学校找过 Eric
穿着女装高跟鞋
晚上牵着 Eric 的手一起逛学校
教 Eric 看星星还教 Eric 长曝光
那一天晚上正好被同学看见
同学还以为是 Eric 女友

时雨
是 Eric 真正意义上的第一个主人
对 Eric 很好
很让着 Eric
也是每天都来找 Eric 玩
会女装但是很少发
能感觉到也有一颗脆弱的心
后来 96子 成为了时雨的主人之后
Eric 也变成了 96子 的宠物
之后叫时雨便改口叫小主人了

96子
是 Eric 现任主人
男友力很高但侵略性也很强
强大的理性派
基本上是 INFP 的死对头
特点就是花心
通过收掉 Eric 当时的主人时雨来收掉 Eric
不过很会关心人
身材很好但是绝不女装
Eric 曾经对 nana 发过誓说死之前一定要看到主人女装照
估计是没戏了
是 Eric 最喜欢的人
也是让 Eric 倾注了一切的人
